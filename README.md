# LOOP zero waste - Интернет-магазин экологичных товаров

Данный проект представляет собой полнофункциональный интернет-магазин, разработанный в рамках курсовой работы по специальности **09.02.07 "Информационные системы и программирование"**.  
**"LOOP zero waste"** — это веб-приложение, ориентированное на продажу товаров для осознанного потребления и образа жизни без отходов (zero-waste).

---

## Основные возможности

Проект представляет собой готовый к развертыванию прототип со следующим набором функций:

- **Каталог товаров** с иерархическими категориями.
- **Поиск и фильтрация** товаров по названию, категориям и товарам со скидкой.
- **Корзина** для авторизованных пользователей и гостей.
- **Избранное** для авторизованных пользователей.
- **Оформление заказа** с выбором способа доставки и оплаты, вводом контактных данных и адреса.
- **Личный кабинет** для просмотра и редактирования профиля.
- **Административная панель** для управления товарами, категориями, заказами, пользователями и настройками магазина.

## Инструкция по установке и запуску

Следуйте этим шагам, чтобы запустить проект на своем локальном компьютере.

### 1. Предварительные требования

Убедитесь, что у вас установлено следующее ПО:
- Веб-сервер: **Apache**.
- PHP: версия **8.x**.
- СУБД: **MySQL 8.x**.
- Браузер: любой современный.

### 2. Размещение файлов проекта

Скопируйте все файлы и папки из архива проекта в корневую директорию вашего веб-сервера.
- Для **OpenServer**: `C:\OSPanel\domains\loop.local\`
- Для **XAMPP**: `C:\xampp\htdocs\loop\`

### 3. Настройка базы данных

1.  Откройте **phpMyAdmin**.
2.  Создайте новую базу данных с именем `loop_eco`.
3.  Перейдите во вкладку **"Импорт"**.
4.  Нажмите **"Выберите файл"** и укажите файл `loop_eco.sql`, который находится в корне проекта.
5.  Нажмите кнопку **"Вперед"** внизу страницы, чтобы завершить импорт структуры и начальных данных.

### 4. Настройка подключения к базе данных

1.  Откройте файл `backend/config.php` в любом текстовом редакторе.
2.  Найдите строки с параметрами подключения и измените их в соответствии с вашими настройками.

### 5. Запуск проекта
Запустите ваш веб-сервер.
Откройте браузер и перейдите по адресу, соответствующему вашему локальному домену.
Если вы использовали папку loop.local: http://loop.local
Если вы использовали папку loop внутри htdocs: http://localhost/loop
Главная страница сайта должна успешно загрузиться.

### 6. Доступ к административной панели

Учетные данные для входа:

- Пароль - admin123
